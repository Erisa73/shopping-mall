#  Shopping Mall – Java Swing Project

##  توضیح کلی برنامه
این پروژه یک **سیستم فروشگاهی (Shopping Mall)** است که با زبان **Java** و با استفاده از **Java Swing** برای رابط گرافیکی پیاده‌سازی شده است.  
هدف پروژه شبیه‌سازی یک فروشگاه آنلاین ساده می‌باشد که در آن کاربران می‌توانند محصولات را مشاهده کرده، جستجو و مرتب‌سازی انجام دهند، سبد خرید داشته باشند و سفارش ثبت کنند.

در این پروژه از **ذخیره‌سازی فایل (CSV)** برای نگهداری اطلاعات استفاده شده و هیچ پایگاه داده‌ای (مانند PostgreSQL) به کار نرفته است.  
پروژه بر اساس اصول **برنامه‌نویسی شی‌گرا (OOP)** و معماری لایه‌ای طراحی شده است.



>  توجه: تمام داده‌ها در فایل‌های CSV داخل پوشه‌ی `data` ذخیره می‌شوند.

## نقش‌های موجود در سیستم

### Customer (مشتری)
- مشاهده لیست محصولات
- جستجوی محصولات بر اساس نام
- مرتب‌سازی محصولات (نام / قیمت)
- افزودن کالا به سبد خرید
- حذف کالا از سبد خرید
- مشاهده قیمت کل سبد
- ثبت سفارش
- پرداخت با استفاده از **موجودی (Balance)**

---

###  Admin (مدیر)
- افزودن محصول جدید
- مدیریت محصولات
- ویرایش اطلاعات کالا
- مدیریت موجودی (Stock)


## کلاس‌های اصلی و مسئولیت‌ها

### models

- **User**
  - کلاس پایه برای کاربران
  - شامل username و password

- **Customer**
  - زیرکلاس User
  - دارای ویژگی `balance` (موجودی حساب)

- **Product**
  - اطلاعات محصول (نام، قیمت، موجودی، وضعیت دسترسی)

- **CartItem**
  - نماینده یک آیتم در سبد خرید (محصول + تعداد)


### services

- **UserService**
  - مدیریت کاربران
  - ورود (Login) و ثبت‌نام
  - ذخیره و بارگذاری کاربران از CSV

- **ProductService**
  - مدیریت محصولات
  - بارگذاری و ذخیره محصولات
  - جستجو و مرتب‌سازی محصولات

- **OrderService**
  - ثبت سفارش
  - بررسی موجودی مشتری
  - بررسی موجودی کالا
  - کاهش Balance مشتری و Stock محصول
  - ذخیره سفارش‌ها در CSV


###  ui

- **LoginFrame**
  - صفحه ورود کاربران

- **ShopFrame**
  - صفحه اصلی فروشگاه برای مشتری
  - نمایش محصولات به‌صورت کارت
  - جستجو، مرتب‌سازی و سبد خرید

- **CartDialog**
  - نمایش سبد خرید
  - حذف آیتم‌ها
  - نمایش قیمت کل به‌صورت زنده

- **AdminFrame**
  - پنل مدیریت محصولات برای مدیر سیستم


## ذخیره‌سازی داده‌ها
در این پروژه از فایل‌های **CSV** برای ذخیره اطلاعات استفاده شده است:
- `users.csv`
- `products.csv`
- `orders.csv`

این روش باعث سادگی اجرا و عدم نیاز به پایگاه داده خارجی شده است.
